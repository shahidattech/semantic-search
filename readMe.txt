This is a under development project, Purpose is to Embed Custom Knowledge Base into Vector database and develop Semantic retrival 
that can be feeded into LLM for generating answers to user Queries, Queries are augmented for better rertival fromEmbedding store. 

Embedding is created using Sentence Transformer through Kubernets Batch Jobs.

This is Full-stack project, frontend is developed using react micro-frontends and Backend is on FastAPI Micorservices.
